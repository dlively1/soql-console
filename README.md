## Abstract

SOQL console web application with auto completion feature.

## Website

[https://soql-console.herokuapp.com/](https://soql-console.herokuapp.com/)

## Help

#### Start Completion
- Tab
- Ctrl + Space

#### Move Cursor to Next
- &#8595; (Down Arrow)
- Tab
- Ctrl + Space

#### Move Cursor to Previous
- &#8593; (Up Arrow)
- Shift + Tab
- Ctrl + Shift + Space

#### Execute Completion
- Return

#### Cancel Completion
- Esc

#### Run Query
- Ctrl + Return
